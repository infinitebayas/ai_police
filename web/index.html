Public-Interest Triage Workspace

AI Police

A support-first assessment workspace for teams that need a flexible developer baseline across local Python, API, container, and GitHub-driven workflows.

Flexible by default

Run it as a Python module, expose it as an API, containerize it, or extend it into other interfaces without changing the core assessment service.

Safe by design

The baseline routes cases toward support, review, and documentation. It does not implement autonomous punitive actions.

Assessment Input

Choose a profile, describe a case, and generate a recommendation.

Assessment Output

JSON output is preserved so contributors can reuse the same result in scripts, tests, or hosted services.
