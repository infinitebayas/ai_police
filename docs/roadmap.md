# Roadmap

## Phase 1: Governance Foundation

- Publish authoritative public-interest documentation.
- Define legal and safety baseline.
- Create jurisdiction adaptation templates.

## Phase 2: Technical Scaffolding

- Define core interfaces and module boundaries.
- Add moderation engine adapter contracts.
- Add retrieval and policy ingestion skeleton.
- Add jurisdiction profile loading and CLI-driven local workflows.

## Phase 3: Oversight and Control

- Implement escalation workflows with approval gates.
- Add audit logging and review dashboard specifications.
- Add rollback and incident response procedures.

## Phase 4: Jurisdiction and Institutional Profiles

- Add region-specific policy packs.
- Add agency deployment profiles.
- Validate legal and procedural alignment per profile.

## Phase 5: Pilot and Hardening

- Controlled pilot with partner institutions.
- Bias, false-positive, and abuse-path testing.
- Harden deployment controls and governance operations.
