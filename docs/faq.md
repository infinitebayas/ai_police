# FAQ

## Is AI Police an official law enforcement body?

No. It is an authoritative AI governance framework that can be adapted by lawful operators and agencies.

## Can this override local laws?

No. Deployments must conform to local law, agency authority, and due process.

## Is full automation recommended?

No. Consequential actions should include human review and accountability controls.

## Can different countries use different moderation standards?

Yes. The framework is designed for jurisdiction-specific adaptation.

## Can public volunteers participate?

Yes, where lawful and governed. Public nodal support should be role-limited, trained, and audited.

## Is this production-ready now?

The current repository is governance and planning-first. Implementation maturity should be validated before production deployment.
