# Release Notes v0.1.0

## Title

Initial public release: governance baseline, API, UI, deployment paths, and optional database support.

## Body

This initial public release includes:

- governance, safety, privacy, and adaptation documentation,
- a support-first Python assessment engine,
- CLI and FastAPI interfaces,
- a lightweight browser UI,
- runtime configuration through `.env`,
- optional PostgreSQL, MySQL, and MariaDB persistence,
- container, Linux VPS, and Azure deployment guidance,
- local verification and GitHub publication runbooks.

## Suggested Short Summary

A local-first, deployment-ready baseline for lawful AI-assisted moderation, auditable triage, and jurisdiction-aware public-interest workflows.
