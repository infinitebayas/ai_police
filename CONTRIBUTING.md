# Contributing to AI Police

Thank you for contributing to AI Police.

This repository supports public-interest governance. Contributions must prioritize lawful operation, transparency, safety, and jurisdictional adaptability.

## Contribution Principles

- Public interest first.
- Lawful scope by default.
- Human oversight is mandatory for enforcement pathways.
- Transparency over hidden logic.
- Auditability and rollback readiness.

## Pull Request Expectations

Each PR should include:

- Summary of problem and intended outcome.
- Scope boundaries and assumptions.
- Legal/jurisdiction considerations.
- Safety and misuse implications.
- Documentation updates for impacted files.

## Required Documentation Updates

If behavior or policy changes, update relevant docs:

- README.md
- docs/architecture.md
- docs/decision-logic.md
- docs/configuration.md
- docs/jurisdiction-adaptation-matrix.md
- PRIVACY.md
- SAFETY.md
- ETHICAL-GUIDELINES.md
- POLICE-DEPARTMENTS-NOTE.md

## Review Criteria

Maintainers review for:

- mission alignment,
- legal boundary clarity,
- operational safety,
- implementation clarity,
- reproducibility of claims.

## Prohibited Contributions

- Extra-legal enforcement claims.
- Hidden political/private suppression logic.
- Unreviewed collection or retention of sensitive data.
- Removal of human oversight points from enforcement paths.
