# Maintenance

## Maintainer Responsibilities

- Keep governance and technical docs current.
- Enforce lawful scope and safety boundaries in reviews.
- Maintain issue triage and release discipline.
- Ensure claims in docs match implementation status.

## Update Cadence

- Monthly documentation consistency review.
- Quarterly policy and safety posture review.
- Immediate updates after critical architecture or risk changes.

## Release Checklist

- README and docs links validated.
- Safety/privacy/jurisdiction docs updated.
- Changelog updated.
- Open critical issues reviewed.
- Risk notes included for major behavior changes.
